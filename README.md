# Comparison Operators
## Description
This project is a simple python program that demonstrates the use of comparison operators. The program compares the two numbers and displays the result.
## Operators Used
- Greater than('>')
- Less than('<')
- Equal to('==')
- Greater than equal to('>=')
- Less than equal to('<=')
- Not equal to('!=')
## Technologies Used
- Pycharm
- Python 3
## How to run
1. Clone the repository.
2. Open the project in pycharm.
3. Run the 'main.py' file.
4. The program compares two numbers.
5. View the result of comparison.
## Sample Outut
```Comparison

10>5=True
10<5=False
10==5=False
10>=5=True
10<=5=False
10!=5=True
```
